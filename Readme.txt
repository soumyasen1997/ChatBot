To run the chatbot, follow the below mentioned steps-

step 1. To create the model, run: python nlu_model.py

step 2. To create the databases of Training Datasets, run:

1. python account_balance_enquiry.py
2. python account_general_query.py
3. python account_money_transfer.py
4. python account_transaction_details.py
5. python card.py
6. python loan.py


step3. To create database pf the User-details, run: python database.py

step4. To vectorize the training datasets, run: python create_tfidf.py

step5. To run the chatbot interface, run: python chatbot.py
